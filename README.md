
# 🍄 Projeto Mario Web

🏠 Projeto Mario Web - Orçamentos Residenciais
Este é um site temático inspirado no universo do Super Mario, desenvolvido para facilitar o contato de clientes que buscam orçamentos para serviços em casa (reformas, reparos, instalações).

A ideia é transformar a experiência de solicitar um serviço em algo visual, lúdico e direto.

🛠️ O que o site oferece?
Interface Temática: Visual nostálgico que torna a navegação mais leve.

Solicitação de Orçamento: Fluxo intuitivo para o usuário entrar em contato.

Responsividade: O site funciona tanto no computador quanto no celular, permitindo que o cliente peça um orçamento de qualquer lugar.

🚀 Tecnologias Utilizadas
Para garantir leveza e rapidez no carregamento, utilizei:

HTML5: Estrutura semântica dos formulários e seções.

CSS3: Estilização personalizada, fontes temáticas e animações dos elementos (nuvens e cenários).

JavaScript: Lógica de validação do formulário de contato e efeitos de transição entre as "fases" do orçamento.
